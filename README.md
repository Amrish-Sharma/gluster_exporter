# gluster_exporter

```
go get github.com/ofesseler/gluster_exporter
./main
```

```
build: Go build
docker: build and run in docker container

``` 

**build**: runs go build for gluster_exporter

**docker**: runs docker build and copys new builded gluster_exporter
